CeSYS je ve zkratce vyhledávač zájezdů a také robustní redakční systém.
Cestovní agentuře nabízí snadno editovatelný web s unikátním designem.
Zároveň je k dispozici administrace pro správu webu a poptávek zájezdů.
\

Moji pracovní náplní zde bylo kódování, tedy
přenesení grafického designu na funkční web.
Zastával jsem zde také roli technické podpory pro klienty CeSYSu.
\

Naučil jsem se tu základy svaté trojice webu HTMl, CSS a JavaScriptu.
Velmi se zde využívala knihovna jQuery pro zrychlení vývoje.
